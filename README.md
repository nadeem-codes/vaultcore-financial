# VaultCore Financial 💳

Enterprise-style Neo-Bank backend application built using Spring Boot and Docker.

##  Tech Stack
- Java 21
- Spring Boot 3
- PostgreSQL
- Docker & Docker Compose
- JPA / Hibernate

---

## Features
- Create User
- Create Account
- Get Account Balance
- Transfer Money
- RESTful APIs
- Dockerized deployment

---

## Run With Docker

```bash
mvn clean package
docker compose up --build

-----
## Application runs at:
http://localhost:8081
